Functions that will be implemented:
  - check_languange_readiness
    This will check if the languange's compiler/interpreter is ready to use.
  
  - compile_code
    As the name suggests, will compile (if possible) all files in a folder.

  - run_code
    Run the compiled code (or interpreting it) with program ID by the parent program.

  - get_IPC
    Get IPC based on the program ID.

  - init_lib
    To initiate the library by internal code.

  - deinit_lib
    To deinitate (freeing) the library by internal code.
